O que é HTML

HTML, que significa Linguagem de Marcação de Hipertexto, é a linguagem base para criar e estruturar páginas web. Ela define o conteúdo e a estrutura de uma página, indicando ao navegador como deve exibir o texto, imagens, links e outros elementos.

Estrutura

O HTML usa tags, que são marcadores que indicam ao navegador o que cada parte do conteúdo representa (ex: um título, um parágrafo, uma imagem, um link, etc.).

Interação com CSS e JavaScript

O HTML funciona em conjunto com outras tecnologias como CSS (para estilização visual) e JavaScript (para interatividade).
