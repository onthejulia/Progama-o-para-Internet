Como subir um arquivo do terminal Windowns

Passo 01:

Ao abrir o prompt de comando inicie abrindo a pasta onde o arquivo está localizado

cd c:\Users\seu nome de usuário\nome da pasta

Em caso de dúvidas em relação ao nome da pasta use o comando "dir"

Passo 02:

Inicialize o git

git init

Passo 03:

Configure passando suas informações de usuário, email e caminho do repositório

git config --global user.name"seu nome de usuário no github"

git config --global user.email"seu email do github"

git remote add origin + url do seu repositório

Passo 04:

Adicione o arquivo desejado

git add nome.extensão

Se desejar subir todos os arquivos da pasta

git add .

Passo 05:

Adicione o comando da branch

git branch -M main

Passo 06:

Suba o arquivo

git push -u origin main

Em casos de erro ao passar o comando push

Nesse caso existem duas opções

Primeira opção

git pull origin main --allow-unrelated-histories

git push -u origin main

Segunda opção

git push -u origin main --force

Obs: esse comando pode subistituir todos os arquivos do repositório. Se faz necessário verificar seu repositório para não ocorrer perda.
